# Keyboard Shortcuts

Shortcut aktif didefinisikan di `src/event/input.h`.

## Global

- `F1`: buka `Help & System Information`
- `F5`: refresh info WinRT
- `Tab`: tampil/sembunyikan inspector
- `Home`: reset camera ke home view

## Camera Presets

- `1`: front view
- `3`: right view
- `5`: isometric view
- `7`: top view

## Scene Toggle

- `Ctrl+G`: toggle ground grid
- `Ctrl+A`: toggle world axes
- `Ctrl+B`: toggle guide cube

## Mouse

- `RMB`: orbit
- `MMB`: pan
- `Mouse Wheel`: zoom

## Catatan

- Saat window kehilangan focus atau mouse keluar dari window, state interaksi kamera dibatalkan untuk menghindari drag yang nyangkut.
